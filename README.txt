-----------------------------------------------------------
 كيف تشغّل الموقع / How to run
-----------------------------------------------------------
الطريقة الأسهل: افتح ملف  index.html  مباشرة بالمتصفح (دبل كليك).

أو كخادم محلي (Windows):
  1) كليك يمين على serve.ps1  >  Run with PowerShell
     أو من PowerShell:  powershell -ExecutionPolicy Bypass -File serve.ps1
  2) افتح بالمتصفح:  http://localhost:5500
